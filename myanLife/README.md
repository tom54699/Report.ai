# myanLife 專案架構圖

## 📖 說明

此 HTML 文件展示 myanLife 多國優惠券與商家預訂系統的完整架構。

## 🚀 使用方式

直接在瀏覽器中開啟 `architecture.html` 檔案即可查看互動式架構圖。

## 📋 內容包含

1. **總覽** - 專案定位與整體架構
2. **CI/CD 流程** - GitLab CI 自動化部署流程
3. **測試服架構** - Linode + Docker Compose 部署方案
4. **正式服架構** - AWS NLB + ALB + ECS + RDS 架構
5. **關聯專案** - 中央後台系統說明

## 💡 特色

- ✅ 互動式標籤切換
- ✅ Mermaid.js 繪製專業架構圖
- ✅ 響應式設計，支援手機瀏覽
- ✅ 清晰的視覺化呈現
- ✅ 無需額外安裝，純 HTML 開啟即用
