📡 A. 串流/直播

💡 核心功能

提供多平台直播推流功能，主播只需推一路流到系統，即可自動轉推到 YouTube、Facebook、Instagram 等多個平台，節省頻寬與操作成本。

1️⃣ 多平台推流

YouTube Live

自動呼叫 YouTube API 建立直播間，取得 RTMPS 推流位址

Facebook Live

整合 Facebook Graph API，支援粉絲專頁與個人帳號直播

Instagram Live

透過 Instagram API 推送直播流（需平台授權）

TikTok Live（待規劃）

TikTok 直播整合功能，依平台 API 開放程度規劃

2️⃣ 技術架構

🏗️ SRS + FFmpeg 架構

  • SRS 媒體伺服器：接收主播的 RTMP 推流，觸發 Webhook 通知
  • FFmpeg 轉推引擎：從 SRS 拉取流並轉推到各平台 RTMPS
  • 按需啟動：每場直播獨立啟動 FFmpeg 實例，直播結束後自動停止
  • 容錯機制：單一平台失敗不影響其他平台

3️⃣ 直播場次管理

功能 說明
建立直播 設定直播標題、描述、封面圖，選擇推流平台
推流設定 自動生成 RTMP URL 與 Stream Key，支援 OBS 或 App 內建推流
直播狀態 READY / STARTING / PUBLISHING / ENDED / FAILED
直播回放 自動保存直播錄影（選配功能）

3️⃣ 關鍵字識別（智能 +1 系統）

🤖 支援多種 +1 格式

  • 基本 +1：「+1」、「+ 1」、「＋1」（支援全形半形）
  • 包裹 +1：「包+1」、「全+1」、「ALL+1」（一次購買全部商品）

4️⃣ 收單流程

📝 從留言到訂單的完整流程

  1. 觀眾留言：「+1 紅色 M」、「我要藍色 L x2」
  2. 系統解析：自動識別商品編號、規格（顏色/尺寸）、數量
  3. 建立暫存單：加入收單池，等待主播確認
  4. 主播確認：檢查庫存、買家資訊，確認或修改
  5. 生成訂單：轉為正式訂單，發送通知給買家

📦 C. 商品管理

💡 核心功能

完整的商品管理系統，支援 SPU/SKU 規格管理、庫存管理、價格策略、候補單機制等功能。

1️⃣ SPU / SKU 管理

📋 商品規格架構

  • SPU（Standard Product Unit）：標準產品單位，例如「秋冬毛衣」
  • SKU（Stock Keeping Unit）：庫存單位，例如「秋冬毛衣-紅色-M」
  • 規格組合：顏色、尺寸、材質等多維度規格
  • 商品編號：用於留言收單時快速識別

2️⃣ 庫存管理

功能 說明
即時庫存 收單後即時扣除庫存，避免超賣
安全庫存 設定庫存下限，低於安全值時發送提醒
批次調整 批量增減庫存，支援 Excel 匯入
庫存歷史 記錄所有庫存異動記錄，可追溯

3️⃣ 價格策略（多層級定價）

💰 靈活的價格分類系統

同一商品可設定多種價格，系統自動根據購買場景與會員等級套用對應價格：

價格類型 說明 使用場景

4️⃣ 候補單（缺貨登記）

💡 當商品缺貨時的解決方案

熱門商品快速售罄時，買家可以登記候補，等商品到貨後系統自動通知，避免流失客戶，並能預估補貨需求。

  • 缺貨登記：買家可登記候補，填寫數量與規格
  • 自動通知：補貨後自動發送通知給候補名單
  • 優先購買：候補客戶享有優先購買權（限時）

自動收單（8 種渠道）
5
系統自動從各渠道收集訂單，解析留言或商城訂單
B. 收單整合

8 種非直播收單渠道詳情：

  • 💬 LINE 群組加單：在 LINE 群組發商品圖片，成員留言「+1 紅色 M」，系統自動收單
  • 👥 FB 社團加單：在社團發商品貼文，成員留言下單，系統自動抓取留言建立訂單
  • 📨 Messenger 私訊：買家私訊詢問商品，聊天機器人自動回應，對話式引導下單
  • 📺 直播回放/貼文加單：直播回放影片留言收單、FB/IG 貼文加單、Reels 短影音收單
  • 🏪 基礎商城：買家瀏覽商城頁面，選擇商品加入購物車，填寫資料結帳
  • 📄 一頁式商城：單一商品推廣頁面，一頁完成下單，快速結帳流程，提高轉換率
  • 🎨 主題式商城：品牌客製化主題商城，專屬視覺設計，完整購物體驗，提升品牌形象
  • 🔗 多平台整合：跨平台訂單匯流（LINE/FB/IG/商城），自動同步庫存，統一後台管理

💡 非直播收單 vs 直播收單的差異

✅ 優點：

• 時間彈性：不需要開直播，24 小時都可以接單

• 渠道多元：8 種收單渠道（LINE/FB/IG/商城等）

• 自動化程度高：系統自動抓取留言/訂單，減少人工操作

• 適合預購/團購：可設定收單期限，統一處理訂單

⚠️ 限制：

• 互動性較低：沒有即時互動，需要透過私訊溝通

• 需要主動宣傳：不像直播有即時曝光，需要主動推廣
